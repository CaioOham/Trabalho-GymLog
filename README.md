# 💪 GymLog — Diário de Treinos

App híbrido para registro e acompanhamento de treinos na academia. Desenvolvido como Atividade de Sistematização da disciplina de Desenvolvimento Mobile.

---

## 📱 Sobre o App

**Problema resolvido:** Praticantes de musculação não têm uma forma simples de registrar e acompanhar seus treinos.

**Público-alvo:** Praticantes de musculação que desejam acompanhar sua evolução.

**Tecnologias utilizadas:**
- React + TypeScript (frontend)
- Node.js + Express (backend)
- SQLite (banco de dados)
- Figma (prototipagem UI/UX)

---

## ✅ Funcionalidades CRUD

| Operação | Descrição |
|---|---|
| **Create** | Cadastrar novo treino com exercício, séries, repetições, carga, data e observação |
| **Read** | Listar todos os treinos cadastrados |
| **Update** | Editar treino existente |
| **Delete** | Excluir treino com confirmação |
